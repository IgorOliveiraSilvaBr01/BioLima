Passos para criar um HTML

  1. Abrir VsCode
  2. Selecionar pastas e arquivos que serão usados
  3. Criar novo arquivo index.html
  4. Montar a estrutura base do documento HTML (usando "!" e "enter")
  5. Definir a linguagem padrão do documento para pt-br
  6. Definir um título ao documento
  7. Usar tag h1 para um título principal
  8. Usar tag h2 para um título secundário
  9. Usar tag p para um parágrafo comum
  10. Execute seu documento no navegador usando "Go Live" para a visualização de tal
